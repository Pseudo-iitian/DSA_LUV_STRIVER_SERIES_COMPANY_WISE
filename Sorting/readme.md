l- Selection Sort
- select minimum and then swap
- TC - O(n^2) which is in the best/worst/average cases.

2- Bubble Sort
- pushes the maximum to the last by the adjacent swaps.
- TC - O(n^2) for worst and average case.
- TC - O(n) for best case.

3- Insertion Sort
- takes an element and place it to its correct order.
- tc :
- 1 2 3 4 5 6 ( best case ) - O(n)
- 5 4 3 2 1 ( worst case ) - O(n^2)
- average tc - O(n^2)

4- merge Sort
- divide and merge
- tc - N * log2(N) : merge sort and merge.
- SC - worst case - O(N) 

5- quick sort
- pick and pivot & place it in its correct place in the sorted array.
- smaller on the left and larger on the right.
- it is also called as divide and conquer method
- tc - O(NlogN)
- Sc - O(1)
